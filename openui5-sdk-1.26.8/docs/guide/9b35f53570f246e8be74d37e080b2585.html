Creating Steps

To create an analysis step, select the category you want to create the step in, click  Add  Step  and enter the following:

Field Explanation
Step Title The title displayed in the analysis step gallery and with the thumbnails in the analysis path display.
Long Title The title displayed above the representation in the analysis step display. If no long title is defined, the title entered in the Step Title field is used instead.
Category The category in which the analysis step is displayed in the analysis step gallery. When first creating a step, this field is already filled with the category in which you create the step. Later on, you can use this field to change the category assignment of a step by selecting a different category from the value help. This is particularly useful if you copy a step and then want to assign it to a different category.
Service The request that defines the data provisioning for the analysis step. Enter the path to the OData service root.
Entity Set Entity set that corresponds to the SAP HANA view.
Properties

The drill-down dimensions, measures, and additional information, for example, the currency, used in the step.

Ensure you select only those properties that shall be displayed in the representation.

Selectable Property The property that is selectable in the chart. If nothing is entered here, nothing can be selected in any representation of this step.
Corner Text for Thumbnail

The texts that are displayed in the four corners of the step thumbnail in the analysis path display. All of them are optional.

Corner texts entered at step level are used as default for the corner texts at runtime as long as no text has been maintained at representation level for the specific corner. They are also used to prefill the corner text fields at representation level.

When you change a specific corner text at step level, this change is reflected in the representation and at runtime only if the corresponding field has not yet been edited on representation level.

At runtime, the steps are displayed in the same order as they appear in the tree structure.
